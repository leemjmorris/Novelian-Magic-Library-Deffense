## 🐛 발견된 이슈 및 개발자 대응

### 🔴 High Priority (긴급 수정 필요)

#### 1. 발사체 관련 이슈
**문제**:
- 발사체가 몬스터와 충돌 시 밀림 현상 발생
- 오브젝트 풀링 시 투사체가 중간에 사라지는 현상
- 투사체가 최초 타겟이 제거될 경우 다른 개체를 유도함

**기획 피드백**:
> "유도하던 타겟이 제거된 경우 해당 타겟을 유도하고 있던 투사체들은 직선으로 날아가는 모습이었으면 합니다."

**개발자 대응**:
- 📅 **2주차 빌드에 수정 예정**
- 몬스터가 동일한 스폰 지역에서 동시에 스폰이 될 경우, 양 옆으로 밀리면서 자리를 잡는 부분에 대한 수정 -> 해당 부분은 스폰 위치를 화면 밖으로 배치하여 해결할 예정
- 투사체가 중간에 사라지는 현상은 현재 발사체의 시작 위치가 Object에서 UI로 이동을 했기 때문에 발생하는 현상. (좌표 문제). 이 부분은 1주차 주간 빌드를 위해 임시로 만들어둔 로직이기 때문에 수정 할 예정.
- 투사체 최초 타겟 제거 시 다른 개체로 유도되는 문제는 해결 되었으나, 투사체가 일직선이 아닌 포물선을 그리며 움직이는 문제 해결 필요. -> 이 부분은 2주차 빌드 내에 해결 목표.

#### 2. 타게팅 시스템 이슈
**문제**:
- 결계에 붙은 몬스터를 캐릭터가 정상적으로 타게팅하지 않고 해당 몬스터보다 상단에 있는 다른 개체를 공격 - 2주차 빌드에 수정 (금일 새벽에 발생한 이슈여서 시간 부족)

**개발자 대응**:
- 📅 **2주차 빌드에 수정 예정**
- 상동.

### 🟡 Medium Priority (중요)

#### 5. 캐릭터 배치 시스템
**문제**:
- 캐릭터 배치 시 그리드의 앞줄 좌측부터 배치된 후 이동 불가
- 랜덤 배치가 아님

**개발자 대응**:
- 📅 **2주차 빌드에 포함 예정**
- 캐릭터의 2D 스프라이트를 UI Canvas 위에서 그리고 있기 때문에 생기는 문제.
- 2D Object화하여 Grid 위치 위에 배치하는 방식으로 수정 예정.

#### 6. 레벨업 UI 이슈
**문제**:
- 레벨업 시 선택 이미지가 표시되지 않음
- 레벨업 카드 이미지 로드 실패

**개발자 대응**:
- 로직은 구현이 되었으나, CSV, 에셋 등이 정해지지 않아서 텍스트만 띄우고 있는 상황.

#### 7. UI Slider Interactive 설정 오류
**문제**:
- 체력바/경험치바 Slider의 Interactive가 활성화되어 있음
- 터치로 체력바나 경험치바를 조절할 수 있는 문제

**개발자 대응**:
- Slider 컴포넌트 설정 수정 필요 (Interactive 비활성화)

### 🟢 Low Priority / 환경 설정 문제

#### 8. 화면 회전 문제
**문제**:
- 게임 실행 시 기기가 Landscape 모드일 경우 게임 화면도 회전하여 UI 비율 깨짐
- 세로 모드 고정이 되지 않음

**개발자 대응**:
- ✅ **해결 방법 확인됨**
- `Default Orientation = Portrait` 환경 설정 변경으로 해결
- 금일 오전부로 수정 완료.

#### 9. 화면 자동 꺼짐 문제
**문제**:
- 게임 실행 중 화면이 자동으로 꺼지는 현상 발생

**개발자 대응**:
- ✅ **해결 방법 확인됨**
- `Screen.sleepTimeout = SleepTimeout.NeverSleep;` 코드 추가로 해결 예정.

#### 10. 플랫폼 호환성
**문제**:
- 태블릿, 폴드 호환 문제.

**개발자 대응**:
- 별도 구현 예정
- 태블릿용과 폴드 전용 UI를 더 만들어두고 분기로 나누어 관리 할 예정.
