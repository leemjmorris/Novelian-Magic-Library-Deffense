- **테스터**:
  - 개발팀 3명 (이명진, 이채빈, 이재문)
  - 기획팀 3명 (김동욱, 김민휘, 김지원)
- **빌드 버전**: 3주차

## 정상 작동 확인 항목

### 전투 시스템
- 전투 기능 전체 작동 완료

### 책갈피 시스템
- 책갈피 제작 기능 구현 완료
- 책갈피 장착 및 확인 기능 구현 완료
- 장착 해제 버튼 구현 완료

### UI
- 스테이지 UI 구현 완료
- 로딩창 구현 완료

### 스킬 시스템
- 스킬 커스터마이징 구현 완료 (현재 하드코딩 방식)

## 발견된 이슈 및 개발자 대응

### 1. 환경 설정 패널 크기 문제
**문제**: 환경 설정 패널이 너무 작아서 사용하기 불편함

**개발자 대응**:
- 담당자: 이명진
ㄴ 환경 설정 패널과 슬라이더의 크기를 키우기로 결정


---

### 2. 캐릭터 이동 조작 문제
**문제**: 캐릭터를 옮길 때 길게 누르고 드래그하는 시간이 너무 오래 걸림

**개발자 대응**:
- 담당자: 이명진
ㄴ 드래그를 담당하는 Input Manager에 해당 값을 인스펙터에서 조절 할 수 있게 만들어서 기획자들이 수정할 수 있도록 제공할 예정

---

### 3. 책갈피 교체 오류
**문제**: 이미 장착된 책갈피끼리 위치를 바꾸면, 한쪽 책갈피가 장착 해제되는 문제 발생

**개발자 대응**:
- 담당자: 이재문
ㄴ 장착된 책갈피들 끼리 위치 교환이 가능하게 만들어야함. 필터도 추가로 제작.

---

### 4. 씬 통합 필요
**문제**: 현재 분리되어 있는 씬들을 하나로 합쳐야 함

**개발자 대응**:
- 담당자: 이명진
ㄴ 주말에 로비 씬에 합칠 수 있는 씬은 모두 합치고, 연출적인 부분은 책장넘기는 연출로 변경할 예정.

---

### 5. 책갈피 제작 후 이동 불편
**문제**: 책갈피를 만든 후 장착 화면으로 바로 이동하는 버튼이 없음

**개발자 대응**:
- 담당자: 이재문
ㄴ 책갈피 제작 완료 패널에 사서 관리로 바로 이동할 수 있는 바로가기 버튼 추가

---

### 6. 이동 중 공격 문제
**문제**: 캐릭터가 이동하는 도중에도 공격이 가능함 (이동 중에는 공격이 불가능해야 함)

**개발자 대응**:
- 담당자: 이명진
ㄴ 캐릭터가 드래그 상태일 때 공격을 멈추도록 로직 추가
